# Codex連携スキル（Zenn記事執筆特化）

OpenAI Codex CLIを使用して、Zenn記事の品質向上をサポートします。

## 実行方法

ユーザーの依頼内容に応じて、以下のコマンドをBashツールで実行してください：

```bash
codex exec --full-auto --sandbox read-only --cd "$(pwd)" "<依頼内容>"
```

## パラメータ説明

| パラメータ | 役割 |
|-----------|------|
| `--full-auto` | 完全自動モード（確認なしで実行） |
| `--sandbox read-only` | 読み取り専用サンドボックス（安全な分析環境） |
| `--cd "$(pwd)"` | 現在のプロジェクトディレクトリを指定 |

## 主な用途

このスキルは以下のタスクに最適化されています：

1. **記事の校閲** - 誤字脱字、文法、技術的正確性のチェック
2. **文章改善** - 読みやすさ向上の提案
3. **画像プロンプト作成** - 記事内容を図解するDALL-E用プロンプト
4. **構成レビュー** - 論理構成の改善提案

詳細な使用例は [examples/usage.md](examples/usage.md) を参照してください。

## 実行時の注意

1. **進捗の可視化**: ターミナル出力がリアルタイムで確認可能
2. **安全性**: `--sandbox read-only` によりファイルは変更されません
3. **結果の活用**: Codexの出力を参考に、Claude Codeで実際の編集を行います

## 呼び出しパターン

ユーザーが `/codex` と入力した場合、何をしたいか確認してから適切なコマンドを実行してください。
